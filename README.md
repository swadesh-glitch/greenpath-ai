# GreenPath AI

## Problem

People struggle to understand and reduce their carbon footprint because existing tools are generic, data-heavy, and not motivating.

## Solution

GreenPath AI is a personalized climate journey platform that combines AI-generated climate identities, actionable missions, future impact simulations, and a living carbon garden to help users make meaningful environmental changes.

## Features

- Climate Identity
- Personalized Missions
- Carbon Garden
- Climate Story
- Future Climate Twin (Phase 2)

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Framer Motion
- Gemini API
- Vercel
