# Phase 1: Core Gamified Experience

Phase 1 focuses on setting up the foundation of the GreenPath AI application, focusing on gamification and a seamless user experience.

## Objectives
- Establish the application structure and styling.
- Create user onboarding flow to select a Climate Identity.
- Implement personalized daily and weekly missions.
- Introduce the Carbon Garden - a visual representation of the user's progress.
- Build the Climate Story and profile dashboards.

## Core Features
1. **Climate Identity & Onboarding**:
   - Assessment questions to determine user persona (e.g., Conscious Commuter, Green Foodie).
   - Character profile card displaying starting traits and specific bonuses.
2. **Personalized Missions**:
   - Interactive checklist of actions across transport, energy, food, waste, and shopping.
   - Micro-animations when completing missions and tracking daily streaks.
3. **Carbon Garden**:
   - Virtual 2D/isometric interactive canvas where plants grow as users complete missions.
   - Visual customization of the garden (colors, themes) reflecting carbon savings.
4. **Climate Story**:
   - A timeline of the user's eco-achievements.
   - Interactive cards highlighting key carbon savings milestones.
