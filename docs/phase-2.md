# Phase 2: AI Core and Advanced Features

Phase 2 introduces the AI brain and advanced simulation models to elevate GreenPath AI into an intelligent eco-advisor.

## Objectives
- Integrate Google Gemini API using `@google/genai`.
- Create the Future Climate Twin simulator.
- Provide dynamic, real-time feedback and smart recommendations.

## Core Features
1. **Gemini API Integration**:
   - **Carbon Chatbot / Coach**: Dynamic responses helping users with specific climate questions.
   - **Smart Mission Generator**: Generates custom missions based on user profile and history.
   - **Garden Narrative**: Generate dynamic descriptions of the garden's status depending on completed missions.
2. **Future Climate Twin**:
   - An interactive simulator illustrating the visual trajectory of the user's local environment by 2050 based on their current footprint.
   - Compare "Business as Usual" vs "Sustainable Future" states.
3. **Advanced Carbon Auditing**:
   - Parse utility bills, receipts, or commuter logs using Gemini's multi-modal capabilities.
